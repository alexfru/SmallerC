Precompiled Smaller C binaries for DOS.
(You may want to use faster DPMI binaries from ../bindp.)
These can be used for cross-compiling for Windows, Linux and MacOS as well.

System requirements:
- DOS: MS-DOS 3.0 or better, i80386 CPU & i80387 FPU or better
- Windows: Windows 95/98/Me/2000/XP or 32-bit Windows Vista/7
- Emulation: DOSBox 0.74 or better
